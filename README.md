<h3>Autonomous agents that securely write, audit, and deploy smart contracts.</h3>

<i>Equipping agents with deep knowledge of Solidity/Rust security patterns, this SKILL.md mandates strict reentrancy checks and mathematical proofs before the agent is allowed to deploy generated code to a testnet, preventing catastrophic hacks.</i>
